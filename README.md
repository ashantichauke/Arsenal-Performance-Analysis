# What Has Changed at Arsenal? - Performance Analysis (Premier League)
## Project Overview
This Business Intelligence portfolio project analyses Arsenal’s Premier League performance across multiple seasons to understand how key performance indicators (KPIs) have changed over time.

The analysis focuses on attacking output, defensive stability, and league competitiveness, identifying patterns in performance and highlighting Arsenal's progression towards consistent top-level finishes.

---

## Business Objective
The aim of this project is to evaluate how performance indicators influence league outcomes and to assess whether improvements in attacking and defensive metrics correspond with stronger league results.

---

## Key Performance Indicators (KPIs)
The following KPIs were analysed:

 - Goals Scored -> measures attacking effectiveness
 - Goals Conceded -> measures defensive stability
 - Points per Season -> measures overall performance
 - League Position -> measures competitve ranking
 - Points per Game (PPG) -> measures consistency
 - Win Percentage -> measures match success rate
 - Top Scorer Goals -> measures individual contribution

---

## Data Preparation
Data was strucutred and prepared in Microsoft Excel to ensure consistency across seasons. 

Calculated fields included: 

- Goal Difference
- Points per Game (PPG)
- Win Percentage
- Season order index for chronological analysis

Only Premier League matches were included to maintain comparability across seasons. 

**Note:**

2025/26 season data refelcts results up to April 2026 and will update as the season progresses.

---

## Dashboard

The Power BI dashboard visualises performance trends across seasons, allowing comparison of attacking output, defensive performance, league position, and consistency metrics. 

---

## Key Insights 

- Goals scored increased significantly between the 2021/22 and 2023/24 seasons, contributing to improved league performance.
- Defensive performance strengthened during peak seasons, supporting higher league finishes.
- Points per Game trends indicate improved consistency compared to earlier seasons.
- League position improved from mid-table finishes towards sustained top-level competitiveness.
- Current season data remains in progress and may change as additional matches are played.
